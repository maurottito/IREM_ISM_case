# IREM ISM Case

Proyecto piloto para digitalizar y revisar formularios de malaria a partir de fotos de hojas en papel.

## Estructura
- `Plan/`: documentación y decisiones del proyecto.
- `scr/`: aplicación web en React + Vite + TypeScript.

## Desarrollo
```bash
cd scr
npm install
npm run dev
```

## Despliegue manual
```bash
cd scr
make deploy
```
